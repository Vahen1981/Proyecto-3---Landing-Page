Proyecto de Landing Page para ToquiControl

ToquiControl es un proyecto personal en el que estoy trabajando, y aprovechando la oportunidad del Proyecto 3, decidí desarrollar una landing page para dicho proyecto.

Esta página incluye el código HTML y una hoja de estilos CSS (styles.css), además de un archivo adicional (styles-mobile.css) para los estilos responsivos mediante media queries.

También he incorporado un archivo script.js para manipular el DOM y así hacer el sitio más interactivo y dinámico.

El sitio fue desplegado en Netlify para su visualización en la web. Puedes verlo en el siguiente enlace: https://landingtoqui.netlify.app/
